# Setting up Visual Studio Code for JavaScript Development

## Install Visual Studio Code

You're not stupid, you can do this without my help.

## Extensions to Install

Click on the Extensions icon in the Activity Bar on the side of the window. Search for the following extensions and click the Install button for each one.

- **IntelliCode** by Microsoft
- **Tailwind CSS IntelliSense** by Tailwind Labs
